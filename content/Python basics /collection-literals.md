```python
#  Dictionary Literals
```


```python
fruits_dict = {
    'a': 'apple',
    'o': 'orange',
    'b': 'banana'
}
```


```python
print(fruits_dict)
```

    {'a': 'apple', 'o': 'orange', 'b': 'banana'}



```python
info_dict = {
    "name":"jerin",
    "age":23,
    "studies":"python"
}
```


```python
print(info_dict)
```

    {'name': 'jerin', 'age': 23, 'studies': 'python'}
